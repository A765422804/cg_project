## 如何运行

项目使用 CMake 进行管理，需确保本地安装的 [CMake](https://cmake.org/download/) 版本 ≥ 3.10

### windows

使用 [minGW](https://github.com/niXman/mingw-builds-binaries/releases) 和 [Clang](https://github.com/llvm/llvm-project/releases) 进行编译，步骤如下
1. 确保已将 MinGW 和 Clang 的 bin 目录添加到系统环境变量 Path 中（避免编译时找不到工具链）。
2. 打开PowerShell，执行以下命令启动构建：

```bash
cd cg_project
.\build_windows.bat
```
